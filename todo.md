# 基本功能

- 显示:
  - 检测结果保存为视频
  - 仅检测显示 机动车、非机动车、行人
  - 使用相机内外参转到俯视坐标显示

- 统计:
  - 统计视频信息: 目标框数量、有目标的帧序号集、动态目标
  - 不同模型检测结果对比
